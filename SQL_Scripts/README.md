# SQL Scripts

This folder contains all SQL queries used for data extraction and analysis in the **Chinook Sales Analysis Dashboard** project.

## 📂 Files
- `Chinook_MySql.sql` – Dataset extraction script that creates and populates the Chinook database in MySQL.
- `Chinoon_Analysis_MySql.sql` – Main analysis queries covering KPIs and insights (Top Customers, Top Genres, Top Artists, Country Sales, Employee Performance, Invoice KPIs).
- `Country_Sales.sql` – Exported query for country‑wise sales visualization in Power BI.
- `Employee_Performance.sql` – Exported query for employee performance dashboard.
- `Top_Artist.sql` – Exported query for top artists by revenue.
- `Top_Genre.sql` – Exported query for genre‑wise sales.
- `Top_Customers.sql` – Exported query for customer revenue ranking.
- `Invoice_KPI.sql` – Exported query for invoice metrics (number of invoices, average invoice value).

## 🎯 Purpose
These scripts demonstrate SQL proficiency in:
- **Joins, aggregations, grouping, and window functions**  
- **Data extraction and transformation** for analytics  
- **Integration with Power BI** for interactive dashboards  

Together, they form the foundation of the Chinook Sales Analysis project, showcasing end‑to‑end data analysis and visualization skills.
