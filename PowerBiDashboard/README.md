# Power BI Dashboard

This folder contains the Power BI dashboard files for the **Chinook Sales Analysis Project**.

## 📂 Files
- `Chinook_Sales_Analysis.pbix` – The main Power BI dashboard file with interactive visuals and KPIs.

## 🎯 Purpose
The dashboard transforms SQL query outputs into **business intelligence insights** through:
- KPI cards (Total Revenue, Average Invoice Value, Number of Invoices)
- Bar charts for customer and artist performance
- Pie chart for genre‑wise sales
- Map visualization for country‑wise revenue
- Employee performance analysis

## 🧰 Tools & Techniques
- **Power BI** – Dashboard design, DAX calculations, KPI cards
- **Data Storytelling** – Professional color palette (Blue, Teal, Orange, Gray)
- **Integration** – Connected to SQL exports for dynamic reporting

## 📈 Outcome
This dashboard demonstrates the ability to:
- Build executive‑grade reports
- Present actionable insights
- Apply visualization best practices for data storytelling
