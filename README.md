# Chinook Sales Analysis Dashboard

## 📊 Overview
This project demonstrates end‑to‑end data analysis using **MySQL** and **Power BI** on the Chinook music store database.  
It transforms raw transactional data into actionable business insights through interactive dashboards.

## 🧩 Project Workflow
1. **Data Extraction (SQL_Scripts/Chinook_MySql.sql)**  
   - Builds and populates the Chinook database in MySQL.  
   - Defines schema, relationships, and inserts sample data.

2. **Data Analysis (SQL_Scripts/Chinoon_Analysis_MySql.sql)**  
   - Performs analytical queries for KPIs and insights.  
   - Generates exports for Power BI visuals (Top Customers, Top Genres, Top Artists, Country Sales, Employee Performance, Invoice KPIs).

3. **Data Visualization (PowerBiDashboard/Chinook_Sales_Analysis.pbix)**  
   - Creates interactive dashboards with KPIs, charts, and maps.  
   - Uses professional color palette and storytelling techniques.

4. **Data Bridge (CSV_Exports)**  
   - Stores exported CSV files used for Power BI integration.  
   - Ensures clean, structured data flow between SQL and Power BI.

## 🧰 Tools & Technologies
- **MySQL Workbench** – Data extraction and analysis  
- **Power BI** – Dashboard design and visualization  
- **Excel/CSV** – Data validation and export  
- **DAX** – KPI calculations and measures  

## 📈 Key Insights
- Top 10 customers contribute over 30 % of total revenue.  
- Rock and Latin genres dominate overall sales.  
- U2 and Metallica are the highest‑earning artists.  
- The USA leads in country‑wise revenue.
